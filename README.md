# 🚀 Fullstack AI Portfolio

A modern, production-ready portfolio application showcasing fullstack development and AI capabilities.

---

## 🧠 Tech Stack

### Frontend

- React + TypeScript + Vite
- Tailwind CSS (planned)
- Framer Motion (planned)

### Backend

- Node.js + Express
- TypeScript (ESM)

### Database

- PostgreSQL (planned)

### AI Services

- RAG (Retrieval-Augmented Generation)
- LLM Integration (OpenAI / Gemini)
- n8n Automation (planned)

---

## 📁 Project Structure

portfolio/
├── frontend/ # React application
├── backend/ # API server
├── ai-services/ # AI pipelines (RAG, automation)

---

## ⚙️ Setup Instructions

### 1. Clone repo

git clone <[repo-url](https://github.com/MMALLIKARJUN2312/My_Portfolio.git)>
cd My_Portfolio

---

### 2. Install dependencies

cd frontend && npm install
cd ../backend && npm install

---

### 3. Run development servers

Frontend:

cd frontend
npm run dev

Backend:

cd backend
npm run dev

---

## 🔐 Environment Variables

Create `.env` files in:

backend/.env
frontend/.env

Example:

PORT=5000
DATABASE_URL=your_postgres_url
OPENAI_API_KEY=your_key
